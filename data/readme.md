# 简介
使用ANN模型来识别纯鹿人是不是IKUN。主要原理就是简单模拟四个基本动作，丢给ANN模型去识别。
## 唱
静止状态下为唱
## 跳
MPU6050水平上下摇晃
## RAP
MPU6050水平左右摇晃
## 篮球
MPU6050全方位摇晃
